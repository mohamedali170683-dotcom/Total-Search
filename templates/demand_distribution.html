Data Source Transparency — What do these numbers mean?

Search Volume (Google, YouTube, Amazon): Actual monthly search queries. Source: DataForSEO API and Google Trends.
Audience Interest (TikTok, Instagram): This is NOT search volume. This number shows the average interactions per post/video about this topic:
  • TikTok: Average likes per video (each like ≈ 1 interested person)
  • Instagram: Average likes per post (each like ≈ 1 interested person)
We use the higher of avg. likes vs avg. comments to avoid double-counting (same person can like AND comment). Higher = more people interested in this topic on that platform.
Interest Index (Pinterest): Relative popularity score (0-100) from Pinterest Trends. 100 = peak interest. NOT an absolute count.
